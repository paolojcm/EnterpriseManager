The EnterpriseManager project is an application responsible for managing information about Enterprises.
The following is a summary of the technology used and the architecture:

Technologies:
* Language: C#.
* Framework: ASP.NET Core Web API(.NET9).
* API Documentation: Swashbuckle OpenAPI (Swagger) documentation.
* Database: SQLite.
* Database engine: DatabaseUtilities.dll
* Logging: Serilog
  
Architecture: Onion Architecture with 3 Layers(Application, Domain and Infrastructure).

Instructions to run:
* Choose a branch.
* Clone the choosen branch.
* Open the solution file EnterpriseManager.API.sln on Visual Studio 2022.
* Press F5 to run it.
